# brain_tumor_analysis
Developed a brain tumor detection system using MRI images with MobileNetV2 transfer learning, data augmentation, fine-tuning, and Grad-CAM for model interpretability. Achieved 88.5% accuracy, 90.2% recall, 90.5% precision, 91.7% F1-score, 96.3% PR-AUC, and reduced false negatives to 4 for reliable tumor detection.
